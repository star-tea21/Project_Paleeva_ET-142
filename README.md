## Курсовой проект: поиск и первичный анализ наборов данных по различным областям применения

В репозитории представлены наборы данных (включая их краткие описания и ссылки на источники), а также программный комплекс на языке Python. Программы предназначены для первичного анализа четырех типов данных: таблиц, временных рядов, изображений и текстов.

**Для каждого типа данных реализованы следующие функции:**

* Загрузка и визуализация;
* Статистический анализ;
* Выявление специфических особенностей структуры данных.

## Описание курсового проекта

Цель проекта — получение практических навыков по поиску и первичному анализу наборов данных разных типов:

* **Табличные данные** — первичный анализ набора данных "Medical Cost Personal Datasets". Направлен на решение задачи регрессии — предсказание годовой стоимости медицинской страховки с учетом демографических факторов и рисков для здоровья.
* **Временные ряды** — исследование почасовых показаний многосенсорных газоанализаторов "Air Quality Dataset" для прогнозирования среднечасовой концентрации угарного газа CО в атмосферном воздухе.
* **Изображения** — анализ репрезентативности и баланса классов набора размеченных фотографий "UTKFace Dataset (Aligned & Cropped Faces)" для задачи бинарной классификации пола человека.
* **Текстовые данные** — очистка и подготовка базы SMS-сообщений "SMS Spam Collection (Text Classification)" для для автоматического обнаружения спама.
